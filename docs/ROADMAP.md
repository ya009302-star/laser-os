# ROADMAP

laser-os は Laser Research OS として段階的にモジュールを追加する。
本ロードマップは方向性であり、確定仕様ではない。
**各バージョンの着手は明示的な指示があるまで行わない。**

## 長期構想 — 研究支援デジタルツイン基盤

laser-os の最終像は、出典をたどれるデータパイプラインによって
物理計算と実験を往復させ、次の実験判断を支援する研究基盤である:

```
論文 (出典)
  → 抽出値 (source_status・出典情報つき)
  → 材料DB (matdb) / 光学素子・光学系DB (optdb)
  → シミュレーション (cavsim: 共振器 / procsim: レーザー加工)
  → 実験ログ (expdb)
  → 予測-実測比較・次実験判断支援 (compare)
```

原則:
- すべての数値は出典までさかのぼれること (定数捏造禁止と表裏一体)
- 予測と実測の差分を記録し続けることがデジタルツインの本体
- 各モジュールは独立パッケージとして src/ 直下に置く (モノレポ)
- 実験生データ・論文 PDF はリポジトリに入れない。リポジトリに入るのは
  コード・スキーマ・書誌情報・参照パスまで (CONTRIBUTING.md 規則 7)

### 予約モジュール名

| 名前 | 役割 | 状態 |
|---|---|---|
| cavsim | 共振器シミュレータ (src/cavsim/) | v0.2 稼働中 |
| litdb | 文献データベース (research/litdb/) | 枠組み設置済み |
| matdb | 材料データベース (research/matdb/, source_status 必須) | 枠組み設置済み |
| optdb | 光学素子・光学系データベース (research/optdb/) | 枠組み設置済み |
| expdb | 実験ログデータベース (research/expdb/) | 枠組み設置済み |
| compare | 予測-実測比較・次実験判断 (research/compare/) | 枠組み設置済み |
| procsim | レーザー加工シミュレータ・解析 | 未着手 |

注: research/ 配下は 2026-06-13 に枠組み (各 README とディレクトリ) のみ
設置。実データは対応研究の進展に応じて段階的に追加する (research/README.md)。
比較・判断は cavsim.analysis (計算コード) との混同を避け compare とした。
matdb は src/cavsim/core/materials.py の最小実装のデータ面の発展先。

### 着手順序の原則

DB やパイプラインより先に**実測照合 1 件**を作る。
比較対象となる実測 (VERIFIED-EXPERIMENT) が存在しない段階で
データベース群を作っても、デジタルツインにはならない。
実測照合の作業を通じて「実験データに何を記録すべきか」が定まり、
それが expdb / optdb のスキーマ設計の根拠になる。

## v0.1 (完了) — cavsim 基盤

- ABCD 共振器シミュレータ (t/s 面、Z-fold、斜入射鏡、ブリュースター結晶)
- 3D ビューポート、スキャン解析、JSON 保存/読込
- コア物理のテスト 9 件 (解析解照合)

## v0.2 (完了: 2026-06-12) — cavsim 物理拡張 + 検証強化

- 非点収差補償角の自動計算 (Z-fold の θ 最適化)
- ブリュースター結晶のビーム横変位の 3D 表示反映
- 分散管理の基礎: 素子ごとの GDD/TOD 記帳 (材料分散はデータ出典必須)
- 熱レンズの単純モデル (薄レンズ近似、焦点距離はユーザー入力)
- 実測モード径との比較ワークフロー (整備済み。VERIFIED-EXPERIMENT 化は
  実測データ提供後: VALIDATION.md §2 の手順)
- 材料データベースの最小実装 (source_status 必須、CONTRIBUTING.md 規則 4)

## v0.2.1 (完了: 2026-06-13) — cavsim 改善 (実測待ち期間)

v0.3 (Research OS 化) には着手せず、cavsim 本体の解析機能を拡張した。
「DB 群より先に実測照合 1 件」の原則は維持 (実測データ待ち)。

- ミスアライメント感度解析 (拡張 3×3 ABCD、ミラー傾き → 各点のビーム移動量)
- 2D スタビリティマップ (間隔 2 つの同時掃引、GUI でクリック設定)
- 安定許容差 (各間隔の不安定化までの距離、二分法精密化)
- 出力ビーム伝搬 (平面端面鏡の外側の w(z)) と z–w コースティックフィット
  (M² 推定; 将来の実測照合で OC 面の w を逆算する準備)

## v0.3 (候補・未着手) — Research OS 化

- 実験ログ管理モジュール (測定条件・結果と cavsim プロジェクトの紐付け)
- 予測-実測比較レポートの自動生成
- 分散/GDD トラッカーの本格化 (共振器往復 GDD、プリズム/チャープミラー)
- モジュール間共通スキーマの整備 (プロジェクト ID、材料参照)

## それ以降 (構想)

- procsim: レーザー加工シミュレータ・解析 (品質目標は cavsim と同じ:
  解析解・文献値・実測との照合を伴う検証主導の開発)
- optdb / expdb / compare の本格化 (上記「長期構想」参照)
- FDTD 系シミュレーション支援 (外部ソルバ連携)

## 不変条件

どのバージョンでも CONTRIBUTING.md のリポジトリポリシーが優先される:
コア/GUI 分離、物理変更へのテスト必須、定数の捏造禁止、
既存挙動の保全、最小差分での変更。
