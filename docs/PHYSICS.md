# cavsim 物理規約 (v0.1)

このドキュメントはコアエンジン (`src/cavsim/core/`) が採用する規約をまとめる。
実装を変更する際は必ずここと整合させること。

## 1. 光線ベクトルと縮約規約

光線ベクトルは **(x, n·u)** を用いる(x: 横変位, u: 傾角, n: 局所屈折率)。
この「縮約規約」ではすべてのABCD行列が **det = 1** となり、
界面での規約の付け替えが不要になる。

- 自由空間/媒質中の伝搬 (幾何長 L, 屈折率 n):

      M = [[1, L/n], [0, 1]]

  つまり媒質中は「縮約長 L/n」の伝搬として扱う。

- 薄肉レンズ (焦点距離 f): M = [[1, 0], [-1/f, 1]]
- 凹面鏡 (曲率半径 R, 凹をR>0): M = [[1, 0], [-2/R_eff, 1]]

## 2. 接線面 (t) / サジタル面 (s)

折返しのある共振器では2つの面を独立に計算する。
本実装では **折返しは水平面内** とし、水平面 = 接線面 (t)、鉛直 = サジタル面 (s)。

### 斜入射凹面鏡 (入射角 θ)

    R_eff(t) = R·cosθ
    R_eff(s) = R/cosθ

接線面の実効焦点距離は f·cosθ、サジタル面は f/cosθ となり、
これが Z-fold の非点収差の主因。

### ブリュースタースラブ (媒質内幾何光路長 ℓ, 屈折率 n)

縮約規約での等価伝搬長(Kogelnik et al. 1972 と等価):

    L_red(t) = ℓ/n³
    L_red(s) = ℓ/n

垂直入射スラブは両面とも ℓ/n。

## 3. ガウシアンビームと q パラメータ

複素ビームパラメータの逆数 1/q を伝搬する:

    (1/q)' = (C + D·(1/q)) / (A + B·(1/q))

縮約規約では媒質によらず

    w² = −λ₀ / (π · Im(1/q))

が成立する(λ₀: 真空波長)。媒質中のレイリー長等を別扱いする必要がない。

## 4. 固有モードと安定性

基準面での往復行列 M_rt = [[A, B], [C, D]] に対し:

- 安定性パラメータ: **m = (A + D)/2**、安定条件 |m| < 1
  (g1g2表記との関係: m = 2·g1·g2 − 1)
- 固有 1/q: 1/q = (D − A)/(2B) ± i·√(1 − m²)/|B|
  (Im(1/q) < 0 となる枝を選ぶ → w² > 0)

t面とs面で独立に判定する。両面とも |m| < 1 のときのみ共振器全体が安定。

## 5. 共振器の表現

- `Cavity(elements, spacings, wavelength)`:
  端ミラー2枚を含む素子列と、隣接素子間の **幾何間隔** [m]。
- 往復: 素子0 → … → 素子N-1 → 折返して逆順 → 素子0。
  端ミラー(素子0, N-1)の行列は往復で1回、中間素子は2回作用する。
- `compute()` は往復行列・安定性に加え、片道経路に沿った w(s) を返す。
  s軸は素子0表面を0とした **幾何座標**(縮約長ではない)。

## 6. 3D配置 (geometry.py)

- 折返し鏡の偏向角: δ = π − 2·(入射角)、`turn = ±1` で左/右折を指定。
- 折返しはすべて水平面 (xy平面) 内。z軸はサジタル方向。
- ブリュースタースラブ (v0.2): 媒質内光線は入射軸から角 φ だけ傾き、
  出射は入射と平行 (平行面)。媒質内パス ℓ に対し

      横変位      = ℓ·sinφ = ℓ(n²−1)/(n²+1)
      軸方向前進  = ℓ·cosφ = 2nℓ/(n²+1)

  (sin²φ + cos²φ = 1 より両成分の合成長は ℓ)。横変位の向きは
  `Crystal.tilt` (±1) で指定し、+1 は cross(進行方向, UP) 方向。
  ABCD 計算には影響しない (縮約長は §2 のまま)。
  検証: tests/test_geometry.py

## 7. 単位

- コアAPI: すべて SI (m, rad)。
- JSONスキーマ (schema_version=1) と GUI: 長さ mm、波長 nm、角度 deg。
  変換は io 層と GUI 層でのみ行い、core には持ち込まない。

## 9. 非点収差補償角 (v0.2)

ブリュースタースラブの縮約光路長差 ΔL = ℓ/n − ℓ/n³ を,
折返し凹面鏡の焦点距離差 Δf(θ) = f/cosθ − f·cosθ (f = R/2) で打ち消す。

第一次近似の釣り合い条件 (`analysis/astigmatism.py`):

    Σ_i Δf(f_i, θ_i) = ΔL

等角・等曲率の k 枚鏡では閉形式

    cosθ = √(1 + M²) − M,   M = ΔL / (k·R)

k=2 が Z/X-fold の標準形 (Kogelnik et al. 1972 系;
レビュー: arXiv:1501.01158 Eq.(9)-(10) と一致)。

`find_compensation_angle()` は閉形式に依存せず、本パッケージの厳密往復行列で
|m_t − m_s| または結晶内 |ln(w_t/w_s)| を数値最小化する。
Z-fold プリセット (ℓ=3mm, n=1.82, R=100mm) で閉形式 6.14° に対し
数値解 6.0–6.4° (指標により異なる) を確認 (tests/test_astigmatism.py)。
閉形式は第一次近似であり、厳密な真円条件とは ~0.2–0.3° 程度ずれうる。

## 10. 熱レンズの単純モデル (v0.2)

`Crystal.thermal_f` [m] を非ゼロに設定すると、結晶中央に焦点距離 f の
薄肉レンズ (t/s 共通) を 1 枚置き、結晶伝搬を半分割して扱う。
縮約長は線形なので半分割の和は全長と厳密に一致する。
f はユーザー入力値であり、本パッケージは熱レンズ焦点距離を推定しない
(ポンプパワー依存の f は実測またはユーザー側の熱解析から与えること)。
f = 0 で無効 (v0.1 と同一の挙動)。負の f (発散) も許容。
検証: tests/test_thermal.py (手組み等価系との厳密一致、f→∞ 連続性、
PropOp↔PathSegment 1:1 対応)。

制限: ブリュースター結晶の熱レンズは実際には t/s で非対称になりうるが、
v0.2 では単一 f の等方モデルのみ (ROADMAP の単純モデル方針)。

## 11. 分散記帳 (GDD/TOD, v0.2)

### 素子ごとの寄与
- ミラー: 1 反射あたり `gdd_fs2` / `tod_fs3` (コーティング値, ユーザー入力)
- レンズ: 1 通過あたり同上 (基板等)
- 結晶/板: 1 通過あたり = 材料分散 (material 設定時) + 追加分 (`gdd_fs2` 等)

### 往復集計 (`analysis/dispersion.py`)
端面鏡 ×1、中間素子 ×2、結晶 ×2 通過/往復。空気分散は無視
(典型共振器長では fs² 未満)。

### 材料分散 (`core/materials.py`)
Sellmeier 式 n²(λ) = 1 + Σ Bᵢλ²/(λ²−Cᵢ) (λ: µm) から

    k₂ = (λ³ / 2πc²) · d²n/dλ²                     [s²/m]
    k₃ = −(λ⁴ / 4π²c³) · (3·d²n/dλ² + λ·d³n/dλ³)   [s³/m]

GDD = k₂·ℓ, TOD = k₃·ℓ (ℓ: 媒質内パス)。λ 微分は 5 点中心差分で評価し、
k(ω) = n·ω/c の ω 直接微分との一致をテストで担保
(tests/test_dispersion.py)。

### データ規律 (CONTRIBUTING 規則 3・4)
材料エントリは source_status と出典が必須。'estimated'/'unknown' の使用時は
警告。v0.2 収録は fused_silica (Malitson 1965) のみ。
**Yb:YAG / YAG は係数の出典指定があり次第追加する** (推定値での登録は禁止)。
`Crystal.material` は分散記帳専用であり、ABCD 計算の n は変更しない
(挙動保全, 規則 5)。
