# Yb:FAP 分光評価 測定プロトコル (SOP)

研究室固有の Yb:FAP 結晶の分光評価。**測定前に記録の型を定めるための文書**
(DATA_SCHEMA の思想: 測ってから形式を決めると項目を取りこぼす)。
本 SOP は ExperimentLog スキーマ (DATA_SCHEMA §7) に準拠する。

> 状態: プロトコル定義済み。実測値・生データは未取得 (これから測定)。
> 生データ (スペクトル/波形 csv 等) は**リポジトリ外保管**、ログからは
> `data_file` で参照のみ (CONTRIBUTING 規則7)。

## ① LD スペクトル (波長校正)

- LD 電流ごとに「904 nm になる設定温度」を求める → **電流-温度対応表**
- 各条件のピーク波長 [nm]、スペクトル幅 (FWHM)、904 nm からのズレ
- 分光器: USB2000、積分時間、平均回数
- ND フィルタ減衰量 (OD 値)
- → quantity: `spectrum` (ld_calibration)

## ② 入力パワー

- LD 電流 [A] vs パワー [W] の表
- **★平均パワーかピークパワーか必ず明記。** QCW デューティ 3% なので
  ピーク = 平均 ÷ 0.03 (≒33倍)。どちらで記録したか曖昧だと後で破綻。
  → PhysicalValue の `condition` に "average" / "peak" を明記する
- パワーメータ: Ophir 1Z01500、波長設定 (904 nm 付近)、
  測定位置 (DM1 の後 = 結晶に入る直前)
- 線形性・高電流側の頭打ち
- (余裕があれば) 結晶透過後パワー → 吸収率
- → quantity: `output_power` (実体は入力ポンプパワー)

## ③ 蛍光スペクトル

- 励起条件 (LD 電流・設定温度・QCW)
- ピーク波長 [nm] (期待値 983 / 1044) と両ピークの強度比
- 集光レンズ (L2) 位置・結晶位置 (再現できるように)
- 分光器: 積分時間、平均回数
- → quantity: `spectrum` (fluorescence)

## ④ 蛍光寿命

- 励起条件 (LD 電流・設定温度・パルス幅)
- **★励起パルスの「立ち下がり後」の自由減衰を測る。トリガは立ち下がり。**
- **★検出器・オシロの応答が寿命 (≈800 µs) より十分速いか/パルス幅 (3 ms)
  が寿命より十分長く定常に達しているか** (応答が遅いと寿命を長く見積もる)
- (留意) 再吸収 (radiation trapping): 厚い/高濃度試料で寿命が見かけ上
  長く出ることがある。気になる場合は励起位置・試料厚で寿命が変わらないか確認
- オシロ: IWATSU DS-5110B、時間軸、平均回数、検出器 (PD 型番・応答速度)
- フィッティング: τ [µs]、フィット範囲、単一指数か複数指数か
- → quantity: `other` (lifetime_decay)

## 出典・データレベルの扱い

- この結晶は研究室固有の作製であり、固有物性の出典は**自分たちの測定**。
  matdb に値を入れる際は `source.type: own_measurement`、
  `data_level: measured`、結びつく実験ログ id を `source` に記録。
- 測定生データは raw、そこから抽出した τ やピーク波長は extracted、
  人間が確認・確定したら verified (CONTRIBUTING の昇格規則)。
